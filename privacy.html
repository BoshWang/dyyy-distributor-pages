Privacy Policy

Last updated: April 7, 2026 · ← Back to home

This Privacy Policy describes how DYYY ("we", "our", or "the application") collects, uses, and protects information when you use the application. DYYY is operated by Kevin Wang as an individual developer.

1. Information We Collect

DYYY is an internal video-distribution tool. It only accesses information that the user explicitly authorizes through OAuth on third-party platforms (such as TikTok, YouTube, Bilibili, and Xiaohongshu). Specifically:

  • OAuth tokens — short-lived credentials issued by each platform after the user grants permission. Tokens are stored locally on the user's own machine and are never transmitted to any third party.
  • Account display name and profile picture — used only to confirm to the user which account is currently authorized.
  • Upload status — when a video is published, DYYY records the platform-returned URL and timestamp locally for the user's reference.

DYYY does not collect, store, or transmit any of the following:

  • User passwords
  • Personal messages, comments, or follower lists
  • Browsing history or analytics from any platform
  • Any data belonging to third-party users (we only handle the connected creator's own content)

2. How We Use Information

The information collected via OAuth is used solely to:

  • Upload videos that the user has produced to the user's own connected accounts
  • Set the title, description, tags, and privacy level of those uploads
  • Display upload status and any error messages back to the user

We do not use the information for advertising, profiling, training machine-learning models, or any commercial analysis.

3. Data Storage and Security

All credentials and metadata are stored exclusively on the user's local machine in encrypted token files (e.g. tiktok_token.json). No data is sent to any cloud server operated by us.

4. Data Sharing

We do not share, sell, rent, or trade any user data with third parties. The only outbound communication DYYY makes is directly to the official APIs of the platforms the user has authorized (e.g. TikTok Open API endpoints).

5. User Control

You may revoke DYYY's access to your account at any time by:

  • Visiting the third-party platform's authorized-apps page (e.g. TikTok Settings → Security → Manage app permissions) and removing DYYY
  • Deleting the local token file from your machine

Once revoked, DYYY can no longer access the account in any form.

6. Children's Privacy

DYYY is intended only for adult content creators and is not designed for use by children under 13. We do not knowingly collect any information from minors.

7. Changes to This Policy

We may update this Privacy Policy from time to time. Updates will be posted on this page with a new "Last updated" date.
